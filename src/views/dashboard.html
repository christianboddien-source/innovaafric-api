Comisiones por Tipo de Transacción

ℹ️ Cómo funcionan las comisiones

Las comisiones se aplican sobre cada transacción. El sistema cobra el porcentaje indicado, con un importe mínimo garantizado. Los cambios se guardan en esta sesión y deben sincronizarse con el backend vía la API de configuración cuando esté disponible.
